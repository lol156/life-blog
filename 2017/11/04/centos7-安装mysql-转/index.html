mysql Ver 14.14 Distrib 5.6.38

先修改一下mysql安装目录下面my.cnf,最后一行添加 skip-grant-tables
然后登陆 进去修改

update mysql.user set password=password(‘mypassword’) where user=’root’;

退出,修改文件删除刚加的属性,重启mysql服务
